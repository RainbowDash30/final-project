Summary Table

Assignment Learned
Chapter 1 In this chapter, I learned the foundations of HTML5, including how to create structured and meaningful content for web pages. I explored essential elements like headings, paragraphs, and lists, and began to understand the importance of semantic markup for accessibility and SEO.
Chapter 2 This chapter introduced me to CSS, where I learned how to style and visually enhance web pages. I explored how to use selectors, properties, and values to control layout, colors, typography, and more, giving web pages a polished and professional look.
Chapter 3 Here, I discovered how to organize content on a webpage using layout techniques like CSS Grid and Flexbox. I learned to create responsive, visually appealing, and user-friendly designs, mastering the art of arranging elements for various devices.
Chapter 4 In this chapter, I focused on using CSS for advanced visual effects, such as gradients, shadows, and transforms. I also learned how to integrate custom fonts and design visually engaging graphics directly in the browser.
Chapter 5 This chapter taught me how to design websites optimized for mobile devices. I explored responsive design principles, including media queries, flexible grids, and touch-friendly interfaces, ensuring that web pages look and function well on all screen sizes.
Chapter 6 Here, I learned how to organize and present data effectively using HTML tables and CSS columns. I discovered how to style tables for readability and create multi-column layouts for better content organization.
Chapter 7 This chapter focused on creating interactive web forms. I learned to build forms with input fields, dropdowns, checkboxes, and buttons while ensuring accessibility and validation for a smooth user experience.
Chapter 8 In this chapter, I explored how to integrate multimedia elements like images, audio, and video into websites. I learned to use HTML5 elements and attributes to create engaging, dynamic web experiences.
Chapter 9 This chapter introduced me to JavaScript, where I learned how to add interactivity to web pages. I explored variables, functions, and events, understanding how to make websites dynamic and responsive to user actions.
Chapter 10 In the final chapter, I delved deeper into JavaScript, learning to work with arrays, loops, and conditional statements. These concepts helped me create more complex scripts, automate tasks, and make logical decisions within my code.
